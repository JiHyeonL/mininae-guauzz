<br>

## 📌 주요 기능
- 조음학습: 조음 발음 영상 & 음성 텍스트 변환 기능으로 학습 
- 낱말학습: 1~15단원의 단어, 문장 발음 영상 & 음성 텍스트 변환 기능으로 학습 
- 복습시험: 각 학습에서 배웠던 내용을 복습하는 객관식/말하기 시험
- 녹음목록: 음성 녹음을 다시 듣고 삭제할 수 있는 목록
- 설정: 글씨 크기, 글씨체 등 설정 가능
